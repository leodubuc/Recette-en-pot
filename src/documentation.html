Documentation vidéo

Description de la composante

Cette composante permet de jouer des vidéo et de changer les miniatures des vidéos.

Dépendances

Il n'y a aucune dépendance

Comment l'utiliser (HTML + CSS)

HMTL

Dans le HTML, il faut ajouter un div avec un data-component et un data-video-id qui sert a savoir quel vidéo jouer

div class="video" data-component="Video" data-video-id="jxCJgTMTUhs"> /div

CSS

Le CSS sert à modifier la miniature de la vidéo.

Tous les paramètres ajustables avec les attributs data

Le data data-video-id sert à savoir quel vidéo jouer

div class="video" data-component="Video" data-video-id="jxCJgTMTUhs"> /div

Le data data-component="Video" sert à cibler la vidéo a partir du javascript

div class="video" data-component="Video" data-video-id="jxCJgTMTUhs"> /div
